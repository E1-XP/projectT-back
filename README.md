## Installation

How to run project locally:

1. Clone repository:
```bash
  git clone git@github.com:E1-XP/projectT-back.git
```
2. Move inside project directory:
```bash
  cd projectT-back
```
3. Install dependencies:
```bash
  npm install
```
4. Run development server:
```bash
  npm start
```
